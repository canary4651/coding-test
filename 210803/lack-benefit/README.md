문제
- [부족한 금액 채우기](https://programmers.co.kr/learn/courses/30/lessons/82612)

## 1. 원하는 것
- 내가 가지고 있는 금액에서 놀이기구를 탈 수 있는 금액이 얼마나 모자란지

## 2. 아는 것
- 놀이기구의 이용료
- 처음 가지고 있던 금액
- 타려는 횟수

## 3. 조건
- 놀이기구의 이용료 price : 1 ≤ price ≤ 2,500, price는 자연수
- 처음 가지고 있던 금액 money : 1 ≤ money ≤ 1,000,000,000, money는 자연수
- 놀이기구의 이용 횟수 count : 1 ≤ count ≤ 2,500, count는 자연수

## 4. 계획
1. 일단 놀이기구 비용에서 count 횟수만큼 값들을 곱해준다음 다 더한다.
2. 그리고 그 총합을 가지고 있는 Money에서 빼준 후 리턴한다.
3. 이때, 마이너스 값이 나온다면 그냥 0를 리턴해준다.
