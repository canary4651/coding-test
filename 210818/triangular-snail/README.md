문제
- [삼각 달팽이](https://programmers.co.kr/learn/courses/30/lessons/68645)

## 1. 원하는 것
정수 n이 매개변수로 주어집니다.
다음 그림과 같이 밑변의 길이와 높이가 n인 삼각형에서 맨 위 꼭짓점부터 반시계 방향으로 달팽이 채우기를 진행한 후, 첫 행부터 마지막 행까지 모두 순서대로 합친 새로운 배열을 return 하도록 solution 함수를 완성해주세요.

## 2. 아는 것
3단계를 거쳐서 삼각형이 완성된다.
1. 맨 왼쪽부터 n번씩 채워진다.
2. 맨 아래에서 오른쪽으로 n-1번씩 채워진다.
3. 맨 아래 오른쪽 위로 n-2번씩 채워진다.
4. n이 1이 될 때까지 반복한다.

## 3. 조건
- n은 1 이상 1,000 이하입니다.

- i를 n만큼 반복하고 j를 n-1번 반복하는 2중 반복문을 작성해야 한다.
1. 아래로 내려가며 배열 채워주기. (i를 3으로 나눈 나머지가 0이면 좌표에 1더하기)
2. 오른쪽으로 옮기면서 배열 채워주기 (i를 3으로 나눈 나머지가 1면 x좌표에 1씩 더해줌)
3. 위로 올라가면서 배열 채워주기 (i를 3으로 나눈 나머지가 2이면 x와 y 좌표를 1씩 빼준다)
